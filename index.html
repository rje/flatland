Flatland

Download this project as a .zip file Download this project as a tar.gz file

What?

Flatland is a simple 2d game engine written in C++ with a javascript front-end. The goal is to develop a system that allows you to quickly iterate on your game ideas and easily package your project for a variety of platforms.

Where?

Initially flatland supports Windows, OS X, and Linux, with iOS and Android also on the TODO list. I'm also eyeing WebGL as a platform target to look at.

Why?

Javascript is one of the most accessible languages out there right now, and it can be a joy to work with. We're trying to provide an environment where people can test out game ideas quickly, and once the ideas are validated they can flesh out their full game within the same environment. Having a minimalistic framework is also great to bootstrap tool development, and I'm hoping the nature of the engine will allow all of us to share tools to make the engine even better.
